# Diplomado IA: ML to LLM

Repositorio de apoyo para clases de Machine Learning con Jupyter Notebooks.

## Estructura

- `01_Taller`
- `02_Regresion_Lineal`
- `03_Gradiente_descendente`
- `04_Regresion_logistica`
- `05_Evaluacion_de_modelos_de_regresion_y_clasificacion`
- `06_Arboles_de_decision`

## Nota sobre ambientes

Este repositorio **no encapsula** ambientes de Python.
Las librerias se instalan y gestionan en ambientes conda locales del instructor o estudiantes.

## Primera etapa

Se incluye un archivo inicial de requerimientos para la Clase 1 en:

- `01_Taller/requirements.txt`

## Convencion de trabajo

- Cada clase tendra sus notebooks `.ipynb` en su carpeta correspondiente.
- Se ignoran checkpoints de Jupyter y archivos del sistema.
